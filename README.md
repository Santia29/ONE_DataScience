# ONE_DataScience

Este proyecto tiene como intención la manipulación y el analisis de datos de cuatro tiendas, los análisis de facturación, ventas por categoría, calificación promedio, productos más/menos vendidos y costo de envío. 
Finalmente, incluirá la conclusión y recomendación sobre qué tienda el cliente debería considerar cerrar, basándose en los resultados obtenidos.
